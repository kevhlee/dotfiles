My personal dotfiles, primarily meant for MacOS.

- Terminal
  - [Kitty](https://sw.kovidgoyal.net/kitty/)
- Shell
  - [Oh My Zsh](https://ohmyz.sh/)
- Package Manager
  - [Homebrew](https://brew.sh/)
- Editor
  - [Neovim](https://neovim.io/)
- Window Manager
  - [Rectangle](https://rectangleapp.com/)

## Usage

To install:

```bash
./dot.sh install
```

To uninstall:

```bash
./dot.sh uninstall
```
